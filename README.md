## Install Jekyll

Ruby must be installed.

1. `gem install jekyll`

---

## Running the kit locally

1. `git clone https://github.com/doaj/doaj-kit.git`
2. `cd doaj-kit`
3. `jekyll s`

---

## Building the website

1. `git clone https://github.com/doaj/doaj-kit.git`
2. `cd doaj-kit`
3. `jekyll build`
4. Content to be published will be found in `doaj-kit/_site`
